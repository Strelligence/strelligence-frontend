# Strelligence Dashboard

Frontend analytics and developer platform for Strelligence.

## Overview

Strelligence transforms raw Stellar/Soroban blockchain activity into structured financial intelligence.

The dashboard provides:
- wallet analytics
- transaction insights
- recurring payment detection
- cashflow visualization
- webhook management
- API key management

## Tech Stack

- Next.js
- TypeScript
- TailwindCSS
- ShadCN UI
- React Query
- Recharts
- Framer Motion

## Features

- Wallet authentication
- Financial dashboard
- Transaction explorer
- AI-generated insights
- Developer API management
- Webhook configuration
